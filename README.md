## 📌 About the Project

This project builds a deep learning model to classify images as either **REAL** or **AI-Generated (FAKE)** using the CIFAKE dataset. The model achieves **93.45% accuracy**, surpassing the original research paper (92.98%).

The project also implements **Explainable AI** using Grad-CAM to visualize which parts of the image the model focuses on when making predictions.

## 🗂️ Dataset

- **Name:** CIFAKE Dataset
- **Source:** [Kaggle](https://www.kaggle.com/datasets/birdy654/cifake-real-and-ai-generated-synthetic-images)
- **Total Images:** 120,000
  - 60,000 REAL (from CIFAR-10)
  - 60,000 FAKE (generated by Stable Diffusion)
- **Image Size:** 32×32 RGB

---

## 🧠 Model Architecture

```
Input (32×32×3)
    ↓
Conv2D(32) → BatchNorm → ReLU → MaxPool → Dropout
    ↓
Conv2D(64) → BatchNorm → ReLU → MaxPool → Dropout
    ↓
Conv2D(128) → BatchNorm → ReLU → MaxPool → Dropout
    ↓
Flatten → Dense(256) → Dropout → Dense(64) → Sigmoid
```

---

## 🔥 Grad-CAM Explainability

Grad-CAM visualizes which regions of the image influence the model's decision:

- **FAKE images** → model detects glitches and artifacts in background
- **REAL images** → model focuses on specific object features
